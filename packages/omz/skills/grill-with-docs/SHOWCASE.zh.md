## 使用场景

当你有一个方案需要接受项目级检验时，用 `/grill-with-docs` 让 Agent 结合项目已有的 CONTEXT.md（领域模型）和 ADR（架构决策记录）对你的方案发起深度挑战。它会用项目自己的术语和已记录的决策来压测你的想法，并在决策明确时同步更新文档。

## 使用效果示例

```terminal
$ /grill-with-docs 我想在 skills 详情页加一个"相关 skills"推荐区域

> 查阅 CONTEXT.md: Skills 是独立可组合的，无层级依赖关系
> 查阅 ADR-003: 详情页保持最小信息架构，不引入侧栏/推荐等次要内容
> ⚠️ 你的方案与 ADR-003 冲突。建议:
>   1. 撤回本方案
>   2. 或先发起 ADR 修订，论证"相关推荐"的合理性
> 请选择方向。
```
