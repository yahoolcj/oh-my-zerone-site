## Use Scenarios

When conversations with the Agent get too long, token usage is high, or you just need a quick confirmation without verbose explanations, use `/caveman` to switch to ultra-compressed communication mode. Agent responses cut ~75% token usage while keeping full technical accuracy. Supports lite / full / ultra / wenyan intensities.

## Example Effects

```terminal
$ /caveman ultra

> CAVEMAN ULTRA MODE.
>
> $ Change skills page? Scope?
> You: Both, SHOWCASE.md replaces SKILL.md
>
> $ List: grid 2-col, scenario summary + code snippet?
> You: Yes
>
> $ Detail: scene prose, effect terminal box?
> You: Yes
>
> $ Filter by category tags?
> You: Yes
>
> $ Go.
>
> [Back to standard] /caveman off
```
