# ISSUE-015 动效 + 响应式收尾

## 父单

.PRD/1.1.0.md — 使用技巧栏目

## 要做什么

完成视觉动效和移动端适配收尾：
1. 调用 `design-taste-frontend` 技能决定 UsageTipsList 和技巧索引页卡片的入场动画方案
2. 实现动效，遵循项目 MOTION_INTENSITY: 6，`prefers-reduced-motion` 时降级为静态
3. 所有新增页面在 mobile（< 768px）下验证布局正确：首页横向列表折叠、索引网格自适应、详情页阅读区

## 验收标准

- [ ] 首页 UsageTips 列表有入场动画
- [ ] 技巧索引页卡片有入场动画
- [ ] `prefers-reduced-motion` 时动画降级为静态
- [ ] mobile 下首页列表行不挤压，可读
- [ ] mobile 下索引页卡片正确折叠（2 列 → 1 列）
- [ ] mobile 下详情页 prose 阅读区宽度合适
