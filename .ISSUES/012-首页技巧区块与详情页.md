# ISSUE-012 首页技巧区块 + 详情页

## 父单

.PRD/1.1.0.md — 使用技巧栏目

## 要做什么

实现端到端技巧展示链路——首页横向列表预览 + 详情页 Markdown 渲染：
1. 新建 `src/components/react-bits/UsageTipsList.tsx`：客户端横向列表组件，每行含序号徽章、标题、摘要（单行截断）、→ 箭头。复用 SpotlightCard 的 hover 交互模式
2. 新建 `src/components/home/UsageTips.astro`：服务端组件，接收 `locale` 和 `limit` prop（默认 6），从 Content Collection 读取 tips 数据，传入 UsageTipsList
3. 新建 `src/pages/docs/tips/[slug].astro`：详情页，通过 `getCollection('tips')` 获取 Astro Content Collection 条目，用 `renderMarkdown` 渲染正文，嵌套在 ProseArticle 中
4. 新建 `src/pages/en/docs/tips/[slug].astro`：同上，locale 为 'en'，使用 titleEn/summaryEn
5. 修改 `src/pages/index.astro`：在 SkillsGrid 后、QuickStart 前引入 `<UsageTips locale="zh" limit={6} />`
6. 修改 `src/pages/en/index.astro`：同上，locale="en"

## 验收标准

- [ ] 首页在 SkillsGrid 和 QuickStart 之间展示 UsageTips 横向列表
- [ ] 列表每行含序号、标题、摘要、→ 箭头
- [ ] 点击行跳转至 `/docs/tips/[slug]` 详情页
- [ ] 详情页通过 ProseArticle 渲染 Markdown 正文，「← 返回技巧列表」链接可用
- [ ] 中英文首页各自工作正常
- [ ] 技巧内容为空时不报错（模板侧优雅处理）
