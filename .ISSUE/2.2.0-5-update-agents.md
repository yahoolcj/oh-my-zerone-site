## 要做什么

更新 `AGENTS.md` 的 Skills 列表，新增 `grill-from-draft` 条目。

当前 Skills 列表中有 12 项（setup-omz 到 to-commit），需在第 2 项 `grill-with-docs` 之后插入第 13 项：

```
N. **grill-from-draft** skill：基于已有需求草稿或设计文档进行深度追问，先补全文档完整性（模糊术语、缺失决策、内部矛盾），再遍历设计树发现新分支，产出 `*_grilled.md` 共识文件。当用户已有初步文档需要打磨澄清时使用。
```

同时检查 `AGENTS.md` 中是否还有其他需要同步更新的引用（如工作流描述部分）。

## 验收标准

- [ ] `AGENTS.md` Skills 列表新增 `grill-from-draft` 条目
- [ ] 条目包含简要功能说明和使用触发条件
- [ ] 编号顺序正确，与已有条目风格一致
